## Static assets

Drop static files served at the site root here.

### Required: `joseph.jpg`

The About section on the homepage references `/joseph.jpg`. Save your portrait
photo as `public/joseph.jpg` (a 4:5 aspect ratio works best — roughly 800×1000
or larger). If the file is missing, a stylized "J" fallback shows in its place.

### Optional: `joseph-intro.mp3`

If you record a ~30-second audio intro and save it as
`public/joseph-intro.mp3`, the homepage About section will automatically
display a play button next to your portrait. If the file is missing, the
component hides itself — no broken UI.
