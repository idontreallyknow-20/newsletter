# Write Better Skill

Review and rewrite the provided text to be clearer, more engaging, and more effective for a newsletter audience.

## Workflow

### 1. Understand the Input

Read the text the user provided. If context is missing, ask:
- What is the goal? (e.g., drive clicks, inform, entertain)
- Who is the audience?
- What tone should it have? (e.g., casual, professional, witty)

If the user pastes text without context, make reasonable assumptions based on surrounding content and proceed.

### 2. Diagnose the Issues

Quickly identify what's weak:
- **Clarity** — Is anything confusing or vague?
- **Concision** — Are there filler words, redundant phrases, or sentences that don't pull their weight?
- **Hook** — Does the opening grab attention immediately?
- **Voice** — Is the tone consistent and appropriate?
- **Structure** — Does it flow logically? Are sentences varied in length?
- **CTA** — If there's a call to action, is it specific and compelling?

### 3. Rewrite

Produce the improved version directly. Apply these principles:

- **Lead with the most interesting thing** — don't bury the lede
- **Cut ruthlessly** — every word must earn its place
- **Use active voice** — "We launched X" not "X was launched by us"
- **Prefer specific over vague** — "3x faster" not "significantly faster"
- **Short sentences for punch, longer for flow** — vary rhythm intentionally
- **Read it aloud mentally** — if it sounds unnatural, rewrite it

### 4. Show Your Work (briefly)

After the rewrite, add a short note (2–4 bullet points max) explaining the key changes you made. Keep this tight — don't over-explain.

### 5. Offer Alternatives (if relevant)

If there's a subject line, headline, or CTA that could go multiple ways, offer 2–3 variants so the user can pick the direction that fits.

## Output Format

```
**Revised:**
[rewritten text]

**What changed:**
- [key change 1]
- [key change 2]
- [key change 3]

**Alternatives:** (optional)
- Option A: [variant]
- Option B: [variant]
```

Keep the revised text clearly separated from the explanation.
